The Art of CSS Design

A demonstration of what can be accomplished through CSS-based design

The Road to CSS Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broken CSS support, and abandoned browsers.

We must clear the mind of the past. Web enlightenment has been achieved thanks to the tireless efforts of folk like the W3C, WASP, and the major browser creators.

The CSS Zen Garden invites you to relax and meditate on the important lessons of the masters. Begin to see with clarity. Learn to use the time-honored techniques in new and invigorating fashion. Become one with the web.

So What is This About?

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation. To begin, view some of the existing designs in the list. Clicking on any one will load the style sheet into this very page. The HTML remains the same, the only thing that has changed is the external CSS file.

CSS allows complete and total control over the style of a hypertext document. The only way this can be illustrated in a way that gets people excited is by demonstrating what it can truly be, once the reins are placed in the hands of those able to create beauty from structure.

Participation

Strong visual design has always been our focus. You are modifying this page, so strong CSS skills are necessary too, but the example files show that even CSS novices can participate if they keep their ambitions modest. Please see the CSS Resource Guide for advanced tutorials and tips on working with CSS.

"The beauty of CSS design is that it separates content from presentation, allowing for endless creativity while maintaining semantic meaning."

Benefits

Why participate? For recognition, inspiration, and a resource we can all refer to showing people how amazing CSS really is. This site serves as equal parts inspiration and learning tool for those working with the web today.

  • Learn modern CSS techniques and best practices
  • Explore creative possibilities within constraints
  • Build a portfolio piece that demonstrates your skills
  • Join a community of passionate web designers and developers
